What I Do ?

Continuously set up new goals, challenges, creating specifications, list of tasks to achieve, simple time calculation of project completion.
After the initial plan has been set, I start to execute each tasks, while I keep track of each modification in some tracking file or in a program for example Git.
Starting from searching for design ideas, templates, continuing with the background logic like how each component should behave in order to successfully provide the requested functionality. I like doing each step.
